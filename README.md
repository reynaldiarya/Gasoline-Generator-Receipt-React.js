# Gasoline Generator Receipt (React.js)

A professional, high-performance React application designed for generating, viewing, and printing gasoline receipts with precision and ease. Built with modern web technologies to ensure responsiveness, speed, and reliability.

## 🚀 Features

- **Dynamic Receipt Generation**: Create itemized fuel entries with automatic calculations.
- **Real-time Preview**: Visualize the receipt instantly as you edit.
- **Print-Ready Output**: High-quality thermal print format support (customizable width).
- **Local Storage Persistence**: Automatically saves your configuration and transaction data locally.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Modern Architecture**: Built with React.js, Vite, and Tailwind CSS for maintainability and performance.

## 🛠 Tech Stack

- **Framework**: [React.js](https://react.dev/) (Vite)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Utilities**: `html2canvas-pro` for image generation.
